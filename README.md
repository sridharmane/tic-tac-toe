# tic-tac-toe
An Implementation of TicTacToe using Plain JS

## [Live Demo](https://sridharmane.github.io/tic-tac-toe/)

## Local Server

```
git clone https://github.com/sridharmane/tic-tac-toe.git
cd tic-tac-toe
npm install
npm start
```
## Build Instructions

```
git clone https://github.com/sridharmane/tic-tac-toe.git
cd tic-tac-toe
npm install
npm build  // for dev
npm run production // for production
```
